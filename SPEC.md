# LED跑马灯 APP 规格说明

## 概念与愿景

一款复古电子风格的LED跑马灯模拟器，模拟真实LED点阵显示屏的质感和光晕效果。界面采用深色主题配合霓虹发光效果，让用户仿佛在操作一台真实的电子显示屏控制器。

## 设计风格

**主题**: 复古电子/霓虹朋克
- 深色背景模拟夜间使用场景
- LED文字带有发光和颗粒感
- 控件采用工业控制台风格，带有金属质感和刻度标记
- 霓虹色彩的交互反馈

## 功能清单

### 核心功能
1. **文字输入** - 支持中英文混合输入，实时预览
2. **文字颜色** - 颜色选择器，提供预设颜色 + 自定义
3. **背景颜色** - 颜色选择器，提供预设颜色 + 自定义
4. **滚动速度** - 滑块调节，范围 20px/s ~ 200px/s
5. **设置保存** - 使用 localStorage 保存最近设置，APP启动时自动恢复

### LED显示效果
- 文字带有LED点阵的颗粒感
- 文字周围带有发光光晕效果
- 平滑滚动动画

### 预设颜色
- 文字: 红/黄/绿/青/蓝/紫/白/橙
- 背景: 黑/深蓝/深紫/深绿/深红

## 技术实现

- 纯HTML/CSS/JS，无框架依赖
- HBuilderX uni-app 或 5+App 兼容
- localStorage 持久化设置
- CSS动画实现滚动效果
- CSS滤镜实现发光效果

## 组件清单

### LED显示屏组件
- 外框: 金属质感边框，带圆角
- 内屏: 深色背景，带内阴影
- 显示: 滚动文字，带LED发光效果

### 控制面板
- 输入框: 深色背景，霓虹边框聚焦效果
- 颜色选择器: 网格布局的预设颜色 + 自定义颜色输入
- 速度滑块: 自定义样式滑块，带刻度标记
- 按钮: 霓虹发光按钮，hover/active状态
